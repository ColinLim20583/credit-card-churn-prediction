# Credit Card Customer Churn Prediction

## Project Overview

This project predicts whether a credit card customer is likely to churn using machine learning techniques.

## Objectives

- Identify customers at risk of leaving
- Analyze factors contributing to churn
- Compare machine learning models
- Provide business recommendations

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib
- Seaborn

## Dataset

Credit card customer churn dataset provided for academic machine learning analysis.
